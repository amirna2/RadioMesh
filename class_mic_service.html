Detailed Description

Service for context-aware MIC computation and verification.

Handles Message Integrity Check (MIC) operations for RadioMesh packets. Selects appropriate keys based on message type and inclusion state:

  • INCLUDE_REQUEST/RESPONSE: ECIES k_mac from ECDH
  • INCLUDE_CONFIRM/SUCCESS: Network key
  • Regular messages: Network key
  • INCLUDE_OPEN: No MIC (returns empty)
