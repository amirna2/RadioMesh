7--------------------------------------------------
8Offset | Size | Description
9--------------------------------------------------
100x0000 | | StorageHeader

14--------------------------------------------------
150x0008 | | First Entry
16 | | EntryHeader
17 | 2 bytes | - keyLength
18 | 2 bytes | - dataLength

20 | N bytes | Key data (length=keyLength)

22--------------------------------------------------
23... | | Next Entry
24 | | (Same structure repeats)
